REACT

React es una biblioteca de JavaScript de código abierto y basada en componentes para construir interfaces de usuario interactivas y reutilizables. Permite la creación de aplicaciones de una sola página (SPA) eficientes, donde los componentes se actualizan de manera eficiente según los cambios en los datos, brindando una experiencia de usuario rápida y fluida.


https://es.react.dev

NODE JS

Node.js es una poderosa plataforma que permite a los desarrolladores utilizar JavaScript tanto en el Front-End como en el Back-End, lo que simplifica la transición entre ambas partes de una aplicación web y brinda un rendimiento eficiente y escalable.


https://nodejs.org/en/about
